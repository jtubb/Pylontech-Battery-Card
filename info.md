# Pylontech Battery Card

A custom Lovelace card for displaying Pylontech (SOK) BMS battery information in Home Assistant.

## Features

- Display battery pack voltage, current, and state of charge
- Show individual cell voltages with visual indicators
- Temperature monitoring for cell groups and components
- System status, protection status, fault status, and alarm status
- Multi-pack support with tabbed interface
- Customizable display options

## Requirements

- Home Assistant 2024.1.0 or newer
- [Pylontech BMS Integration](https://github.com/jtubb/HA-Pylontech-BMS) installed

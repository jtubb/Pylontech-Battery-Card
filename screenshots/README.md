# Screenshots

This directory contains screenshots for the README documentation.

## Files

- `battery-overview.png` - Main battery system overview card
- `temperature-heatmap.png` - Interactive temperature heatmap view
- `voltage-heatmap.png` - Interactive voltage heatmap view
- `history-graph.png` - Mini-graph-card history view (optional)

## Adding Screenshots

1. Take screenshots of your battery card in Home Assistant
2. Save them to this directory with the names above
3. Commit and push to update the README images

## Image Guidelines

- **Format**: PNG (preferred) or JPG
- **Max Width**: 1200px (will be displayed smaller in README)
- **File Size**: Keep under 500KB each for faster loading
- **Content**: Show actual battery data with clear, readable text
